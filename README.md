# Carp Co-occurrence Analysis

This repository contains analysis of carp co-occurrence in fish communities. Please view the following Jupyter notebooks for the full analysis:

1. **[ltrm_fish_summary.html](ltrm_fish_summary.html)** – Summary and exploration of LTRM fish data and explanation of co-occur analysis.
2. **[ecosimr_lcode.ipynb](ecosimr_lcode.ipynb)** – Ecological simulation of community level co-occurence by lcode and modeling using EcoSimR R package.
3. **[ecosimr.ipynb](ecosimr.ipynb)** – Ecological simulation of community level co-occurence by strata and modeling using EcoSimR R package.
4. **[coocur.ipynb](coocur.ipynb)** – Probability-based co-occurrence analysis of carp with other fish species using cooccur R package.
