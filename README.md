# Carp Co-occurrence Analysis

This repository contains analysis of carp co-occurrence in fish communities. Please view the following Jupyter notebooks for the full analysis:

1. **[ltrm_fish_summary.ipynb](ltrm_fish_summary.ipynb)** – Summary and exploration of LTRM fish data.
2. **[coocur.ipynb](coocur.ipynb)** – Co-occurrence analysis of carp with other fish species.
3. **[ecosimr.ipynb](ecosimr.ipynb)** – Ecological simulation and modeling using EcoSimR.
